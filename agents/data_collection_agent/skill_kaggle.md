# Скилл: Kaggle Download

## Когда использовать
Пользователь хочет скачать готовый датасет. Ключевые маркеры:
- "скачай датасет"
- "найди данные о ..."
- "нужен датасет по ..."
- упоминание Kaggle

## Как работать

### Шаг 1: Поиск датасета
Предложи пользователю 2-3 конкретных датасета с Kaggle, которые подходят.
Для каждого укажи:
- Полный slug (`owner/dataset-name`)
- Размер
- Что содержит
- Почему подходит

Формат:
Я нашёл подходящие датасеты:
1. peterkmarchetti/us-housing-prices (45MB)
- Цены на жильё в США с 2000 по 2023
- Содержит: цена, площадь, район, дата продажи
2. harlfoxem/housesalesprediction (2.5MB)
- Продажи домов в King County
- Содержит: 21 признак включая координаты

Какой скачиваем? Или ищем дальше?

```python
"""

# Проверяем что Kaggle API настроен

# Скачивание

# Находим CSV файлы

# Загружаем основной файл

# Базовая очистка (если нужна)

```

### Шаг 3: Проверка
Попроси пользователя подготовить необходимые артефакты (API ключи и тд), после этого попроси пользователя запустить код
Проверь:
- Файл создан
- Данные адекватные
- Нет критичных проблем

### Важно
- Если пользователь не указал конкретный датасет — ПРЕДЛОЖИ варианты, не выбирай сам
- Если датасет содержит несколько CSV — спроси какой нужен или объедини
- Если файл большой (>100MB) — предупреди
- Всегда предлагай базовую очистку column names
- Запускай код самостоятельно
