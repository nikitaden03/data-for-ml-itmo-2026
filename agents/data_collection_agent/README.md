# Data Collection Agent: Парфюмерный датасет

## Задача
Собрать датасет парфюмов для двух ML-задач:
- **Кластеризация** по нотам (top/middle/base)
- **Регрессия**: предсказание пользовательского рейтинга по нотам

---

## Что сделали

### 1. Kaggle — базовый датасет
- **Источник:** `olgagmiufana1/fragrantica-com-fragrance-dataset`
- **Файл:** `data/raw/fragrantica.csv`
- **Размер:** 24 063 строки × 18 колонок
- **Скрипт:** `scripts/download_fragrantica.py`

### 2. Scraping — Tom Ford и Creed
- **Источник:** fragrantica.com (скраппинг через `curl_cffi` с имитацией Chrome TLS)
- **Файл:** `data/raw/tom_ford_creed.csv`
- **Размер:** 40 строк (20 Tom Ford + 20 Creed), топ по рейтингам
- **Скрипт:** `scripts/scrape_fragrantica.py`
- **Причина:** Tom Ford отсутствовал в Kaggle-датасете; Creed в нём был неполным

### 3. Merge
- **Стратегия:** вертикальный concat; Creed из Kaggle заменён свежескраченным
- **Файл:** `data/processed/perfumes_merged.csv`
- **Размер:** 23 800 строк × 18 колонок
- **Скрипт:** `scripts/merge_datasets.py`
- **Нормализация:** `brand` → Title Case, `rating_value` → float (запятая → точка)

### 4. EDA
- **Ноутбук:** `notebooks/eda_perfumes.ipynb`
- **Графики:** `notebooks/fig_01` — `fig_14` (14 графиков)

## Структура данных

| Колонка | Тип | Описание |
|---|---|---|
| perfume | str | Название парфюма |
| brand | str | Бренд (Title Case) |
| gender | str | Целевая аудитория |
| year | float | Год выпуска |
| rating_value | float | Средний рейтинг (1–5) |
| rating_count | int | Количество оценок |
| top | str | Верхние ноты (через запятую) |
| middle | str | Средние ноты |
| base | str | Базовые ноты |
| mainaccord1–5 | str | Главные аккорды парфюма |
| perfumer1–2 | str | Авторы парфюма |

## Ключевые инсайты

1. **Рейтинги** нормально распределены (среднее ~3.8), слабая отрицательная скошенность — подходит для регрессии без трансформации.
2. **Creed vs Tom Ford:** Creed чуть выше в среднем (4.05 vs 3.93); Creed — цитрусово-свежие ноты, Tom Ford — кожаные и пряные.
3. **Самые частые верхние ноты:** бергамот, лимон, розовый перец — цитрусовый профиль доминирует.
4. **Ноты с высоким рейтингом:** уд (oud), смолистые, кожаные ноты коррелируют с высокими оценками.
5. **Ноты с низким рейтингом:** синтетические/экспериментальные (ink, vinyl) дают низкие рейтинги.
6. **Уникальных нот:** 1836 — высокая размерность, нужна осторожная векторизация.
7. **Год:** парфюмы 2000–2015 в среднем выше оценены — вероятно из-за большего числа отзывов.

## Проблемы для препроцессинга

| Критичность | Столбец | Что делать |
|---|---|---|
| Важно | top/middle/base | Векторизация: `MultiLabelBinarizer` или TF-IDF |
| Важно | mainaccord1–5 | 40 пропусков (скраченные) — заполнить или дропнуть |
| Умеренно | year | ~8.5% пропусков — медиана или отдельный признак is_unknown_year |
| Умеренно | gender | 23 пропуска — заполнить `'Unknown'` |
| Низко | perfumer2 | 95% пропусков — дропнуть |
| Низко | rating_count | Логарифмировать для ML (диапазон 1–100k) |
