# Data Collection Agent — Парфюмерный датасет

## Задача

Собрать корпус русскоязычных отзывов на парфюмы и обогатить его структурированными данными (ноты, рейтинг) со scraping'а Fragrantica — для дальнейшего NLP-анализа и задачи рекомендаций.

---

## Ключевые инсайты

1. **By the Fireplace (Maison Margiela) — лидер по рейтингу (4.21/5)** при самом большом числе голосов на Fragrantica (24 439), что говорит о высокой репутации в мировом сообществе. При этом в русском корпусе он второй по числу отзывов (203).

2. **Chanel No 5 — самый обсуждаемый в русском корпусе (208 отзывов)**, но при этом имеет наименьший рейтинг (3.60/5). Это указывает на поляризацию мнений: классика вызывает споры у современных потребителей.

3. **Средняя длина отзыва существенно различается**: Chanel No 5 (575 симв.) и Cuoium (552 симв.) получают самые развёрнутые рецензии — сложные, неоднозначные ароматы провоцируют более детальное обсуждение. The One Gentleman (300 симв.) — самые короткие.

4. **Bergamot — наиболее универсальная нота**: присутствует в 5 из 8 парфюмов (Chanel No 5, By the Fireplace, Black Pearls, Acqua di Parma Colonia, Aspen For Men). Vanilla — вторая по частоте встречаемости.

5. **Динамика отзывов охватывает 2009–2025**: всплеск активности в 2023–2024. By the Fireplace и Cuoium привлекли основной поток отзывов в этот период — тренд на нишевые/дымные ароматы.

6. **Cuoium (Orto Parisi) — нишевый лидер**: минимальное количество нот (кожа, ладан, кедр), высокий рейтинг (3.96), большое количество развёрнутых отзывов (201). Показатель того, что "сложные" ароматы привлекают увлечённую аудиторию.

## Проблемы для препроцессинга

| Проблема | Критичность | Рекомендация |
|----------|-------------|--------------|
| Малый размер датасета (931 строка, 8 парфюмов) | **Высокая** | Расширить: скачать больше парфюмов / источников |
| Ноты хранятся строкой через запятую, не нормализованы | **Высокая** | Разбить в список, привести к единому регистру |
| Кодировка файла (`utf-8-sig`) требует явного указания | **Средняя** | Стандартизировать на UTF-8, задокументировать |
| Дата отзыва — строка, не datetime | **Средняя** | `pd.to_datetime(format="%d.%m.%Y")` |
| Пропуски в `review_text` (NaN в исходнике) | **Средняя** | Фильтровать или заполнять пустой строкой |
| Fragrantica scraping нестабилен (Cloudflare) | **Средняя** | Добавить retry-логику и кэширование HTML |
| Дублирование нот в merged (каждый отзыв повторяет ноты парфюма) | **Низкая** | Хранить ноты отдельно и джойнить при необходимости |
| Английские ноты при русских отзывах | **Низкая** | При необходимости перевести или использовать как есть |
