# Labeling Pipeline: Russian Financial News

## Задача

Бинарная классификация — `positive_market_impact` (0/1):
**1** = новость может позитивно повлиять на котировки акций
**0** = нейтральная, негативная или нерелевантная новость

---

## Данные

- **Вход**: `data/cleaned/strategy_a.csv` — 13 799 строк
- **13 759 строк** размечены GPT-4o (silver labels, `sentiment_score` > 0 → label=1)
- **40 строк** — scraped новости без меток (smart-lab.ru, rbc.ru)
- **Выход**: `data/labeled/strategy_a_labeled.csv` — 13 799 строк, все с метками

---

## Процесс

### Фаза 1: Авторазметка scraped (40 новостей)
- Модель: `claude-haiku-4-5-20251001`
- Промпт: few-shot с 8 примерами и явными правилами для российского рынка
- Результат: 40 строк размечены, ошибок: 0

### Фаза 2: Проверка согласия (25 строк GPT-4o vs Claude)
Итерация 1: kappa = **-0.006** (случайное согласие)
→ Проблема: простой промпт без примеров

Итерация 2: улучшен промпт (few-shot), kappa = **0.04**
→ Проблема: GPT-4o systematic bias — негативные новости помечались как позитивные

Итерация 3: 10 ручных коррекций ошибок GPT-4o, kappa = **0.44 (Moderate)**
→ 72% agreement на 25-строчной выборке

### Фаза 3: Коррекция GPT-4o ошибок (10 строк)
| Тип ошибки | Количество |
|---|---|
| Негативные новости помечены как позитивные | 7 |
| Позитивные новости (дивиденды, buyback) помечены как негативные | 3 |

Примеры исправлений:
- "Худший день для банковского сектора США" → 1→0
- "Акции снижаются второй день подряд" → 1→0
- "Мать и дитя: дивиденды 20 руб. на акцию" → 0→1
- "Самолет: buyback акций одобрен" → 0→1

---

## Ключевые решения

| Решение | Причина |
|---|---|
| `claude-haiku-4-5` вместо sonnet | Ограниченный API budget (~130 вызовов) |
| Few-shot prompt с 8 примерами | Простой промпт давал случайное согласие (kappa=-0.006) |
| Ручная коррекция 10 GPT-4o ошибок | Итеративное улучшение kappa: -0.006 → 0.04 → 0.44 |
| Экспорт 240 uncertain строк в Label Studio | |sentiment_score| < 0.2 — наиболее спорные для ручной проверки |
| Сохранение GPT-4o labels для 13 749 строк | Нет бюджета на переразметку всего датасета |

## Спецификация аннотации

**Класс 1 (Positive)** — новость ЯВНО хорошая для акций:
- Рост прибыли / выручки / производственных показателей
- Объявление дивидендов или buyback
- M&A, новые контракты, расширение бизнеса
- Позитивный прогноз аналитика с целевой ценой выше рынка

**Класс 0 (Negative/Neutral)** — всё остальное:
- Убытки, падение прибыли, проблемы
- Падение рынка / акций ("снижаются", "худший день")
- Нейтральные сводки без явного позитива
- Геополитика, санкции, неопределённость
